## 👀 게임 설명
로또게임 입니다.

### 🎯 &nbsp;구현 기능 목록
#### UI
- [x] 구입할 금액 입력 받기
- [x] 구매 결과 보기
  - [x] 토글시 레이아웃 변경
- [x] 지난 주 당첨번호 ~ 결과 확인하기 버튼

#### 도메인
- [x] 단위 테스트를 작성한다.
- [x] 금액을 입력받는다.
  - [x] 구매 금액은 1000원으로 나누어 떨어져야 한다.
  - [x] 금액은 1000원 이상 10000원 이하여야 한다.
- [ ] 구입 버튼을 누르면 구매금액 / 1000의 수만큼 랜덤한 숫자를 뽑는다.
 - [ ] 랜덤 숫자는 1번부터 45번 사이의 숫자이다.
 - [ ] 랜덤 숫자는 중복되지 않는다.
 - [ ] 랜덤 숫자는 6개 뽑는다.
